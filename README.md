# LibraConnect – Simple Library Borrowing Website
##  Project Overview

LibraConnect is a multi-page library website built with HTML, Bootstrap, and JavaScript. we collaborated by creating individual pages with unique JS features, including DOM manipulation and interactive functionality.

## Key Features by Page
1. Home Page
  - Render featured books using JS array and Bootstrap cards
  - Toggle "Read More/Less" descriptions with addEventListener

2. Catalog Page
- Display books dynamically in cards
- Live search by title with keyup event

3. Register/Login Page
- Form fields: Name, Email, Password
- Validation: email must contain @limu.edu.ly, password ≥6 characters, simple Captcha
- Prevent submission if validation fails

4. Borrow Page
- Display books with checkboxes
- Show message when a book is selected
5.  History Page
- Render borrowing history from JS array
- Toggle extra info (return date, fines) using classList.toggle()
